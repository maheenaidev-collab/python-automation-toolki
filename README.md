# 🐍 Python Automation Toolkit

A collection of powerful Python scripts for automation, web scraping, and data processing.

**Built by Maheen — AI & ML Engineer**

---

## 🚀 Scripts Included

### 1. Web Scraper (`web_scraper.py`)
- Scrape data from any website
- Export to CSV or JSON
- Customizable tag targeting

### 2. File Automator (`file_automator.py`)
- Organize files by extension automatically
- Bulk rename files
- Find duplicate files

### 3. Data Processor (`data_processor.py`)
- Load & clean CSV datasets
- Statistical analysis (min, max, avg)
- CSV to JSON converter
